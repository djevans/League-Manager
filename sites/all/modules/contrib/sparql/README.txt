
SPARQL for Drupal 7.x
=====================
This is a development version of the SPARQL API currently being developed
for Drupal 7.x under the auspices of the project at:

  <http://drupal.org/project/sparql>


BUG REPORTS
-----------
Post bug reports and feature requests to the issue tracking system at:

  <http://drupal.org/node/add/project_issue/sparql>
